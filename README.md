**Project Overview:**

PlateShare is a MERN full-stack application that allows users to share surplus food within their community. Users can post food items for donation, browse available foods, and submit requests. The app includes Firebase authentication, a MongoDB database, and hosted images via imgbb.

---

## Key Features

Firebase Authentication (Email/Password + Google login)
 
 CRUD operations for food management (Add, Update, Delete, View)
 
 Food request system for donors and recipients
 
 Dynamic Featured Foods and responsive grid 

 Loading states and 404 error 

 Animations with Framer Motion / AOS / react-spring (optional)
